Chat session Code Specifications:

When I say WIDGET("exampleName", "newPropertyName")
I mean that you should look for the example template code between comments like these blocks, 
depend on the file type, HTML or Javascript, as follows:

HTML source file has this block in it somewhere : 

    <!-- WIDGET-EXAMPLE("exampleName", "property", "property1", "property3" ... ) -->
        Sample code is here.
    <!-- END-WIDGET-EXAMPLE("exampleName") -->

    
In the chat I would then say WIDGET("exampleName") which means you should look
    for the template text in between the WIDGET-EXAMPLE("exampleName") comment 
    and the END-WIDGET-EXAMPLE("exampleName") comment and generate code with the rest of the 
    arguments I supply in the call to WIDGET().

Javascript source file has this block in it somewhere : 

    // CODE-EXAMPLE("exampleName", "property1", "property2", "property3" ... )
        Sample code is here.
    // END-CODE-EXAMPLE("exampleName") 

In the chat I would then say CODE("exampleName") which means you should look
    for the template text between the CODE-EXAMPLE("exampleName") comment
    and the END-CODE-EXAMPLE("exampleName") comment, and generate code using the rest of the 
    arguments I supply in the call to CODE().
